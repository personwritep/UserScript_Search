「UserScript Search」は「UserScript List」の姉妹ツールです。
「UserScript List」の機能から、スクリプト名の検索抽出機能のみを取り出し、専用ツールにしています。
このツールの特徴は以下です。

◎ 常駐使用を考慮して「Ctrl+F10」の専用のショートカットを用意
◎ リスト表示用データをストレージ保存し、起動直後に検索機能が使える
◎ 検索絞り込みにのみ使用するリスト1個の小型ウインドウで起動
